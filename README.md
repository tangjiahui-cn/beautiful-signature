## beautiful-signature

A js library for hand-signature, which support any web platform.

## Features
- Support Platform: Mobile, Web.
- Support pen tip handWriting.
- Custom pen color.
- Custom pen size.
- Custom pen type. Built-in: Signature-Pen，Pen，Brush, Color-Pen, etc.
- Custom pen.
- Custom renderer.
